# ⚡ 快速开始指南 - 5 分钟上手

在 5 分钟内启动 Project Aegis,并录制一个 30 秒的演示视频发给面试官。

---

## 📋 步骤 1: 启动应用 (2 分钟)

### Windows 用户
1. 双击 `启动.bat`
2. 等待浏览器自动打开 (约 30 秒)
3. 看到 "🛡️ PROJECT AEGIS" 标题 → 成功!

### Mac/Linux 用户
```bash

cd ~/Desktop/ByteDance_SpringFestival_Ops_Toolkit_Ian

# 安装依赖 (首次运行)
pip install -r requirements.txt

# 启动应用
streamlit run app.py
```

## 📋 步骤 2: 熟悉界面 (2 分钟)

### 快速浏览清单
- [ ] 看到倒计时: "距离除夕: XX 小时"

1. **SPS 仪表盘** (在 "物流热力图" Tab 左侧)
   - 指针在 3.6 附近抖动
   - 红色区域 = 危险 (< 3.5)

2. **Smart+ 熔断器** (第二个 Tab)
   - ROAS 曲线 (绿色 = 正常,红色 = 熔断)
   - Spend Velocity 图表 (烧钱速度)

3. **NRR Sniper** (第三个 Tab)
   - 输入框: 输入 "Shipping took forever!"
   - 点击 "🚀 AI 分析"
   - 看到结果: "📦 Logistics (Appealable)"

4. **SPS 监控大屏** (底部)
   - 筛选器: 选择 "仅 P0 Critical (< 3.5)"
   - 看到红色高亮的店铺
   - 点击 "🚨 触发飞书报警" 按钮

## ⚠️ 常见问题

### Q1: 启动后浏览器没有自动打开?
**A**: 手动访问 `http://localhost:8501`

### Q2: 报错 "ModuleNotFoundError: No module named 'streamlit'"?
**A**: 运行 `pip install -r requirements.txt`

### Q3: 数据不刷新?
**A**: 这是正常的,数据每 60 秒自动刷新一次。你也可以按 `R` 键手动刷新。

### Q4: 想修改店铺数量 (从 500 改为 1000)?
**A**: 编辑 `app.py`,找到 `generate_shop_data(500)`,改为 `generate_shop_data(1000)`

### Q5: 如何停止应用?
**A**: 在终端按 `Ctrl+C`,或直接关闭终端窗口

## 🎓 进阶技巧

### 技巧 1: 录屏时添加鼠标高亮
- Windows: 使用 OBS Studio + 鼠标高亮插件
- Mac: 使用 Mouseposé (付费) 或 KeyCastr (免费)

### 技巧 2: 添加背景音乐
- 使用剪映 (免费) 或 Premiere Pro
- 推荐音乐: YouTube Audio Library (免版权)
- 音量: 20-30% (不要盖过人声)

### 技巧 3: 添加字幕
- 使用剪映的 "自动识别字幕" 功能
